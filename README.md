# Shaik Mohammad Abdullah — Portfolio

A clean, accessible product engineer portfolio built with Next.js, React, and Tailwind CSS.

## Features

- **Responsive** — Works on all screen sizes
- **Dark/Light mode** — Toggle with persistence
- **Accessible** — Semantic HTML, ARIA labels, focus states
- **Subtle animations** — Framer Motion for scroll and hover effects

## Setup

1. Install dependencies: `npm install`
2. Add your resume PDF to `public/resume.pdf` (for the Download Resume button)
3. Run dev server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy to Vercel, Netlify, or any static/hybrid host. Remember to add `resume.pdf` to the `public` folder before deploying.
